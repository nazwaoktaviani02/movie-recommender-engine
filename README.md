# Movie Recommender Engine 

Hi Globees~  
This is a simple content-based recommendation system built with **Python**. It's designed to find similar movies based on their metadata like genres, cast, director, and keywords. 

I built this as part of my journey to **rewire my brain** through code and storytelling. You can read the full breakdown of how this works on my [https://medium.com/@itsnaz/bongkar-cara-kerja-movie-recommender-engine-pakai-python-simple-banget-df0d2a3302ab].

---

## Features
- **Content-Based Filtering**: Recommends movies similar to the ones you love.
- **Natural Language Processing (NLP)**: Uses `CountVectorizer` to turn text into math.
- **Similarity Scoring**: Uses `Cosine Similarity` to find the perfect match.

## Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-Learn

## Dataset
The system uses a movie dataset containing:
- `Keywords`, `Cast`, `Genres`, and `Director`.
